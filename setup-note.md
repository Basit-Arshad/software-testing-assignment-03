# Setup Note

## Project Type
Custom Java-based order processing system developed for JUnit 5 testing.

## Environment Configuration
- IDE: IntelliJ IDEA Community Edition
- JDK Version: 17
- Build Tool: Maven
- Testing Framework: JUnit 5
- Coverage Tool: JaCoCo

## Project Import and Setup Steps
1. Created a Maven project in IntelliJ IDEA.
2. Configured the project to use JDK 17.
3. Added JUnit 5, Apache POI, and JaCoCo dependencies in pom.xml.
4. Created the main Java classes inside the `com.example.order` package.
5. Created unit test classes inside the `src/test/java/com/example/order` package.

- Project compiled successfully.
